Introduction

Welcome to svg playground
This app serves as a reference application on using svg for the web.
At the bottom of this page you can find a summary of svg elements currently available.
You can also go to mozilla docs for more detailed information.
The links provided here is just a filtered list that excludes depreciated features of features not yet implemented.

The filters are separated from the elements as they are basically about pixel manipulation used on the other elements.

This application has several screens that you can find in menu.
These screens are samples that serve as a reference.
These references will grow over time.
